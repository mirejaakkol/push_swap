# Libft

My own C library: a from-scratch implementation of commonly used standard functions, plus extra helpers I reuse across projects.

## Contents

- Character checks and conversions (`ft_is*`, `ft_to*`)
- String utilities (`ft_strlen`, `ft_strlcpy`, `ft_strjoin`, `ft_split`, ...)
- Memory utilities (`ft_memset`, `ft_memcpy`, `ft_calloc`, ...)
- File descriptor output helpers (`ft_put*`)
- Bonus: linked list utilities (`ft_lst*`)

## Build

```bash
make
make bonus
```
